## 12. Architectural decisions worth knowing

(Verbatim or paraphrased from `DECISIONS.md` + `CLAUDE.md`. Don't unwind without flagging.)

- **No backend.** Everything is static JSON + GitHub Pages. No database, no auth, no server cron ‚Äî CI cron only.
- **No personal-brand framing.** Site reads as a project, not a person.
- **No forward-looking privacy promises** ("we don't / never / ever").
- **No fake contact paths** ‚Äî don't add `hello@borderpulse.com` until that inbox exists.
- **Embeds are `noindex`** to avoid SEO competition with canonical detail pages.
- **No new third-party scripts** (analytics, email, etc.) without asking.
- **Bilingual EN/ES is hand-rolled** ‚Äî no i18n library. Pattern: `const strings = { en: {...}, es: {...} }; strings[language].key`.
- **Code-split everything except Dashboard.** Main bundle target ~161 KB.
- **Scheduled `chore(data)` commits skip the deploy workflow** to avoid double-runs.
